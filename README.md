# Sistema Bancario

Proyecto desarrollado en **Java** que simula el funcionamiento básico de un sistema bancario.

## Funcionalidades

* Registro de clientes
* Creación de cuentas bancarias
* Transferencias entre cuentas
* Gestión de préstamos
* Bitácora de operaciones

## Estructura del proyecto

El sistema está compuesto por varias clases que representan las entidades del banco:

* ClientePersonaNatural
* ClienteEmpresa
* CuentaBancaria
* Transferencia
* Prestamo
* BitacoraOperacion

## Tecnologías utilizadas

* Java
* Programación Orientada a Objetos (POO)
